# Lipi Anniversary — Part III

A self-contained, responsive Part III of the anniversary story website.

## Included
- `index.html`
- `style.css`
- `script.js`
- all supplied Cloudinary photo URLs embedded directly in the page

## Run locally
Open `index.html` in a browser, or use:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Hosting
Upload the contents of this folder directly to Netlify, Cloudflare Pages, or GitHub Pages.

## Notes
- The countdown targets **24 June 2026, 12:00 AM IST**.
- The ambience toggle creates a small original in-browser ambient pad only after the visitor taps it.
- The site uses Google Fonts and Cloudinary-hosted photos, so it needs internet access once hosted.
